# Contributing to AI Tools Hub

Thank you for your interest in contributing to AI Tools Hub! 🎉

## How to Contribute

### Suggesting New AI Tools

The easiest way to contribute is by suggesting new AI tools:

1. **Check if the tool already exists** in our directory
2. **Open a GitHub Issue** with the label `new-tool`
3. **Provide the following information:**
   - Tool name
   - Website URL
   - Category (Text, Image, Video, Audio, Code, Productivity, Business, Other)
   - Brief description (1-2 sentences)
   - Pricing model (Free, Freemium, Paid)
   - Rating (if available)

### Reporting Bugs

Found a bug? Help us fix it:

1. **Check existing issues** to avoid duplicates
2. **Create a new issue** with label `bug`
3. **Include:**
   - Browser & OS version
   - Steps to reproduce
   - Expected vs actual behavior
   - Screenshots (if applicable)

### Suggesting Features

Have an idea for improvement?

1. **Open an issue** with label `enhancement`
2. **Describe:**
   - The problem you're solving
   - Your proposed solution
   - Why it would benefit users

### Translating

Want to add a new language?

1. Contact us via email or GitHub issue
2. We'll provide the translation template
3. Submit your translations as a Pull Request

### Improving Documentation

- Fix typos in README
- Clarify confusing sections
- Add examples
- Update outdated information

## Code of Conduct

- Be respectful and inclusive
- No spam or self-promotion
- Stay on topic
- Help create a welcoming environment
